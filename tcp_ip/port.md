## 端口 Port
```
OSI分层结构中的每层都有一个唯一标识
数据链路层  MAC
网络层     IP地址
传输层     Port

TCP用2字节的整数来表示端口，一台主机最大允许65536个端口号
```

### 端口号分类
```
熟知端口号: 0~1023，熟知端口号由专门的机构IANA分配和控制，如果要监听这些端口号需要Root权限(保留端口)
已登记端口号: 1024~49151，已登记端口不受IANA控制，由IANA登记并提供它们的使用情况清单，为什么是49151呢? 49151=65536*3/4 - 1，占了大约75%
临时端口号: 49152~65535 (动态端口、私有端口)，如果没有调用bind()函数将socket绑定到特定端口，TCP或UDP会为该socket分配一个唯一临时端口
           不同操作系统实现会选择不同的范围分配临时端口，在linux上能分配的端口范围由/proc/sys/net/ipv4/ip_local_port_range 变量决定
           cat /proc/sys/net/ipv4/ip_local_port_range
           32768  60999
```
