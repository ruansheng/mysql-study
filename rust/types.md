## 原生类型
```
原生类型是内建在语言中的
```

### bool 布尔类型
```
Rust有一个内建的布尔类型，叫做bool，有2个值，true 和 false
let x = true;
let y:bool = false;
```

### char
```
char类型代表一个单独的unicode字符，可以用(')单引号来创建
let c = 'x';
let d = '';
这里的c或d不是1个字节，而是4个字节
```

### 数字类型
```
Rust有一些分类的大量数字类型: 有符号和无符号、定长和变长、浮点和整型

有符号:
i8
i16
i32
i64

无符号:
u8
u16
u32
u64

可变大小类型:
isize
usize

浮点类型:
f32
f64
```

### 数组
```
数组是一个定长相同的元素列表，数组默认是不可变的
数组的类型是 [T;N]
这里的N是编译时常量，代表数组的长度

let n = [1,2,3];  // [i32;3]
let m = [4,5,6];  // [i32;3]

初始化数组:
let s = [0,30];  // s中的每个元素都被初始化为0

获取数组长度:
s.len()

用下标访问元素:
s[1]
```

### 切片
```
一个切片是一个数组的引用，在底层，slice代表一个指向数据开始的指针和一个长度

切片语法:
可以用一个&和[]的组合从多种数据类型创建一个切片

let a = [1,2,3,4,5];
let b = &a[..]
let c = &a[2..4]
```

### str
```
str是Rust最原始的字符串类型
```

### 元组 Tuples
```
元组是固定大小的有序列表
let x = (1,"hello");

let y:(i32,&str) = (1,"hello");

元组可以赋值，要求是类型和元素数量都一致
let x = (1,4);
let mut y = (2, 3)
y = x;

解构元组中的元素:
let (x,y,x) = (1,2,3);

元组索引:
let x = (1,4,7);
let a = x.0;
let b = x.1;
```
